# VetSystem
Sistema de Gestión para Clínica Veterinaria

## Descripción
Proyecto Final de Programación Orientada a Objetos - UTP

## Tecnologías
- Java JDK 25
- JavaFX 21
- MySQL 8.0
- Maven
- NetBeans

## Características
- Gestión de pacientes (mascotas)
- Gestión de dueños
- Registro de consultas médicas
- Control de medicamentos
- Uso de Iterator, ListIterator y Lambda
